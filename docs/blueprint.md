# **App Name**: GoldenEye Trades

## Core Features:

- Live Price Feed: Continuously fetch and display the current XAU/USD price from a reliable API. 
- Signal Generation: Generate BUY/SELL trading signals for XAU/USD based on real-time prices and technical analysis, leveraging the models to produce trading signals, and making the signals available as a tool
- Signal Monitoring: Automatically monitor active trading signals and detect when they hit take-profit or stop-loss levels. Store the signals' monitoring state and hit detection
- User Management: Admin panel for managing user accounts, roles, and permissions.
- Historical Signals: Display a list of completed trading signals, including entry price, take-profit levels, stop-loss, and outcome (win/loss), saving the trade history
- Authentication: Secure user authentication with bcrypt for password hashing.
- Admin Interface: A dashboard to view and manage users, create users, manage user roles (admin/user), and control user status (active/inactive).

## Style Guidelines:

- Primary color: Rich gold (#D4AF37) to reflect the value and nature of gold trading.
- Background color: Soft beige (#F5F5DC) to provide a warm, neutral backdrop.
- Accent color: Deep brown (#8B4513) for important actions and highlights.
- Headline font: 'Playfair', a serif font to give an elegant, high-end feel.
- Body font: 'PT Sans', a sans-serif font, paired with Playfair, to maintain readability and modernity.
- Use professional, crisp icons for trading actions and status indicators.
- Subtle animations for real-time price updates and signal changes to enhance user experience.
